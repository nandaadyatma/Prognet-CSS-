Apa Itu CSS?


CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah.


Dengan CSS, kita dapat mengatur layout (tata letak), warna, font, garis, background, animasi, dan lain-lain.



Nah, untuk lebih jelas kita bisa lihat pada gambar di atas, yang mana pada gambar kiri merupakan halaman website yang menggunakan CSS sedangkan yang kanan merupakan halaman website tanpa menggunakan css. Dari gambar tersebut kita sudah bisa melihat perbedaanya jelasnya, yang mana pada halaman yang menggunakan CSS (gambar kiri) lebih memiliki tampilan yang menarik :D


 

Bagian-bagian CSS




 

Selector

Selector berfungsi untuk memilih element mana yang hendak kita beri style. Selector bisa dituliskan dengan nama tagnya, misal: h1, b, div, p, dll. Bisa juga dituliskan dengan nama class-nya. selector class diawali dengan tanda (.)


Nama Tanda Contoh Penjelasan
tag nama-tag h1, h2, p, div, ul Mengatur styling pada elemen seperti <h1>, <h2>, <p>, <div>, >ul<, dan lainnya


 

Property

Ada banyak sekali jenis property pada CSS. property ini berfungsi untuk mengatur sebuah elemen, apakah mau elemen mau diberi warna latar, warna teks, shadow, dan lain-lainnya.

Begitu banyak pilihan property css yang bisa kita gunakan. Tetapi kita tidak akan dibahas disini satu persatu karena ada banyak sekali. Beberapa contoh property yaitu:


Nama Fungsi
color Untuk mengubah warna teks
background-color Untuk mengubah warna latar belakang teks
font-size Untuk mengubah ukuran teks


 

Value

Value merupakan nilai yang dimasukkan ke dalam property. Value juga bisa beragam bentuknya. Bisa berbentuk angka, hexacode, teks string biasa. hingga opsi-opsi khusus yang terbatas


Jenis Contoh Keterangan
Angka 20px, 2rem, 1em, 100% Biasanya digunakan untuk mendefinisikan suatu ukuran.
Warna black, #FFF, rgba(0,0,125,1) Digunakan untuk mendefinisikan warna.

Contoh penggunaan property

  h1 { color: red; }

  p {
  font-family: 'Courier New', Courier, monospace;
  color: blue;
  text-align: justify;
  }

Cara Menulis CSS


Terdapat 3 cara yang dapat kita lakukan untuk menulis CSS pada HTML.

1. Inline CSS

2. Internal CSS

3. External CSS

 


Inline CSS

Pertama ada inline style sheet. Sederhananya, CSS model ini adalah CSS dengan perintah pemrograman yang letaknya ada pada elemen.

Jadi dengan cara ini kita menuliskan style yang kita inginkan di dalam tag elemen tersebut. Seperti contoh berikut

  ...
  <h2 style="color: yellow;"> Ini Judul dengan H2 </h2>
  <p style="color: yellow;">Lorem, ipsum dolor sit amet consectetur.
  Maiores reprehenderit impedit doloribus fugit magnam magni quo?
  Quos dolores, quibusdam sed nostrum cumque autem adipisci quaerat
  tempore inventore sapiente optio corrupti.</p>
  ...

 


Internal CSS

Kemudian ada internal CSS. Sama seperti inline CSS model ini sama-sama diletakkan di dalam file HTML, namun perbedaanya dari peletakannya. Dengan cara ini kita bisa meletakkan style di dalam tag <head>... </head>. Style internal CSS ini ditulis di dalam tag <style>... </style>

Nah, untuk lebih jelasnya bisa lihat contoh penulisan di bawah


  <!DOCTYPE html>
  <html>
  <head>
   <title>Contoh Internal CSS </title>
     <style>
     p {
      font-family: serif;
      line-height: 1.75em;
     }
     i {
      font-family: sans;
     }
          </style>
  <body>
  ...

 


External CSS

Nah cara yang terakhir yaitu, dengan cara external CSS. External CSS ini adalah kode CSS yang ditulis terpisah dengan kode HTML. Eksternal CSS ditulis di dalam file yang berbeda, yang mana file ini berekstensi .css.

Contoh kita membuat file 'inicss.css' yang di dalam berisi kode CSS seperti berikut.


    p {
      font-family: serif;
      line-height: 1.75em;

     }

    i {
     font-family: sans;
    }

    h1 {
     font-family: sans;
     color: orange;
    }



Untuk menggunakan CSS ini, kita harus menghubungkannya dengan File HTML.

Ada dua cara yang bisa kita lakukan:

  1. Menggunakan tag `<link>`
  2. Menggunakan `@import`

 

Box Model CSS



Pada CSS, ada yang istilah yang namanya "Box Model". Box model ini digunakan ketika berbicara tenntang tata letak dan desain


Box model pada CSS pada dasarnya merupakan sebuah kotak yang membungkus tiap elemen pada HTML. Pada box model, kita akan mengenal beberapa bagian yaitu margin, border, padding, dan content


  • Content - Isi kotak, tempat teks dan gambar muncul
  • Padding - Jarak transparan diantara content dan border
  • Border - Batas yang mengelilingi padding dan content
  • Margin - Jarak trasnparan di luar border

Contoh penggunaanya seperti di bawah ini


    div {
     width: 400px;
     border 2px solid blue;
     padding 20pt;
     margin 30pt;
    }
